# Public FAA Reference Sources

CertFlow AI uses synthetic demo certification cases. No proprietary Collins Aerospace, RTX, Boeing, FAA internal, or customer-confidential data is used.

The workflow and terminology are inspired by public FAA certification planning materials and public airworthiness compliance references.

## Public sources used for realism

### 1. FAA Project Specific Certification Plan Template

The FAA PSCP template describes the structure of a Project Specific Certification Plan and supports the idea of planning certification projects around certification basis, compliance methods, deliverables, schedules, and responsibilities.

Source:
https://www.faa.gov/sites/faa.gov/files/aircraft/air_cert/design_approvals/dah/pscp_template.docx

### 2. FAA Enhanced Project Specific Certification Plan Guide

The ePSCP guide includes an example compliance checklist structure with regulations, method of compliance, compliance documents, and designees.

Source:
https://www.faa.gov/sites/faa.gov/files/aircraft/air_cert/design_approvals/dah/ePSCP_guide.pdf

### 3. FAA AC 23-24 Airworthiness Compliance Checklists

AC 23-24 provides information for creating compliance checklists for common Part 23 Supplemental Type Certificate projects.

Source:
https://www.faa.gov/documentLibrary/media/Advisory_Circular/AC_23-24.pdf

### 4. FAA Form 8110-12

FAA Form 8110-12 is used for applications related to Type Certificates, Production Certificates, Supplemental Type Certificates, and amended certificates.

Source:
https://www.faa.gov/forms/index.cfm/go/document.information/documentID/1043079

### 5. FAA Certification Project Planning References

FAA certification project planning references describe planning elements such as product description, schedule, certification basis, means of compliance, communications and coordination, delegation, test plans, documentation, production certification, and post-certification requirements.

Source:
https://www.faa.gov/aircraft/air_cert/design_approvals/transport/how_to_plan/why_create_plan

## Why this matters

Aerospace certification work is document-heavy, evidence-driven, exception-heavy, and human-review-dependent. CertFlow AI demonstrates how UiPath Maestro Case can coordinate AI agents, automation steps, and human certification reviewers in a governed enterprise workflow.
